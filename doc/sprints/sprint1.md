## Sprint 1: The Foundation
**Goal:** Establish the technical infrastructure and core visual identity.

* **Project Setup:** * Configure `vite.config.js` to handle the multi-page structure within `src/pages/`.
    * Initialize `package.json` with necessary scripts and dependencies to resolve previous build errors.
* **Global Styling:** * Create a `main.css` file implementing a strict CSS Grid to ensure all text and images feel "locked" into a mathematical Bauhaus structure.
    * Define CSS variables for the Triadic Palette representing the three acts: Yellow (#F5D328), Pink (#F2A2B1), and Black (#1A1A1A).
    * Set typography to Sans-serif (Helvetica or Futura) with heavy weights.
    * Establish global rules for Square, Circle, and Triangle UI elements, strictly avoiding rounded corners on containers.
* **Title Page (`index.html`):** * Implement a minimalist hero section featuring a full-screen background video and a high-contrast geometric entry button leading to the home page.
* **Global Navigation:** Build the persistent minimalist header and the "Call to Action" footer for all pages.
